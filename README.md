# Gold Price Prediction

This project predicts next-day gold price (GLD) using market indicators.
Files:
- app.py : Streamlit app
- gold_model.joblib : trained model
- scaler.joblib : feature scaler
- feature_cols.joblib : ordered feature list
- gld_price_data.csv : dataset
- requirements.txt

To run locally:
1. pip install -r requirements.txt
2. streamlit run app.py
